# BBA Facility Services

A modern React website for BBA Facility Services - Professional facility management and property services in Switzerland.

## Services Offered

- **Hauswartung** (Facility Management)
- **Gebäudereinigung** (Building Cleaning)
- **Baureinigung** (Construction Cleaning)
- **Gartenarbeit** (Garden Work)
- **Umbau** (Renovation)
- **Concierge-Service**

## Features

- Modern, responsive design
- Smooth scrolling navigation
- Interactive contact form
- Mobile-friendly layout
- Beautiful gradient hero section
- Service cards with hover effects
- Professional logo with building icon

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
